# `@guardian/prettier`

> Prettier config for Guardian JavaScript projects.

## Installation

```bash
yarn add -D @guardian/prettier
```

or

```bash
npm install --save-dev @guardian/prettier
```

## Usage

```yaml
# Prettier configuration file
'@guardian/prettier'
```

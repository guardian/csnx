# QuoteIcon

The Guardian quote icon is used in kickers, headlines, drop caps and pull quotes to denote quoted text.

## Install

```sh
$ pnpm add @guardian/source-development-kitchen
```

or

```sh
$ npm i @guardian/source-development-kitchen
```

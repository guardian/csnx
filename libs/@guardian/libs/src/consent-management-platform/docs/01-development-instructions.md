# Development instructions

## Testing

### e2e Tests

From the csnx directory run

```sh
make e2e
```

### Github Pages

From the csnx directory run

```sh
make github-pages:dev
```

then go to the 'CMP Test Page' on the top left or directly to [CMP Test Page](http://localhost:4322/csnx/cmp-test-page#tcfv2)
